    App criado na linguagem Python para o Projeto de extensão 
da matéria: Tópicos de Big Data em Python do 4º período do
Curso de Análise e Desenvolvimento de Sistema - EAD da Faculdade Estácio de SÁ
Pólo Neópolis - SE

Foram utilizadas as bibliotecas "Pandas" e "Plotly" para o desenvolvimento do App,
o arquivo requirements.txt incluso na documentação do App contém todas as 
bibliotecas e dependências necessárias ao funcionamento do App, que podem ser
instaladas atravéz do comando "  pip install -r requirements.txt ".

A ambiente de desenvolvimento uitilizado para programar e testar foi o Visual Stúdio Code 
com o Python3.12.4 e para visualização dos Data frames foi utilizado o Navegagor web 
Edge Versão 127.0.2651.98 (Compilação oficial) (64 bits) com  SO Windows 11 PRO.

O Aplicativo ler os dados de um arquivo CSV e gera Data Frames com os seguintes resultados:

PIB_BRASIL 2000 A 2021
PIB_SERGIPE 2000 A 2021
PER_CAPITA_BRASIL 2000 A 2021
PER_CAPITA_SERGIPE 2000 A 2021
COMPARAÇÃO PER_CAPITA_BRASIL VS PER_CAPITA_SERGIPE 2000 A 2021

O objetivo da criação do App é poder verificar a evolução do crescimento do PIB do Brasil e 
estado de Sergipe entre os anos de 2000 a 2021, bem como a evolução de PIB Per Capita em 
Sergipe e no país no mesmo período e comparar como essa evolução aconteceu para podermos 
verificar se o crescimento do PIB Per Capita do estado de Sergipe seguiu a mesma trajetória de 
crescimento do Pib Per Capita do Brasil.
